# Maize Pest Knowledge Base (MaizeKB)

A comprehensive knowledge base and interactive **Streamlit app** for maize pests. This app provides detailed pest profiles, their damage effects, plant parts affected, and recommended control methods for both field and storage pests. 

---

## Features

- Detailed **pest profiles** including:
  - Common name & scientific name
  - Pest type (Insect, Nematode, Vertebrate)
  - Pest category (FieldPest, StoragePest, VertebratePest)
  - Damage summary and description
  - Plant parts attacked
  - Crop stages affected
  - Control actions and general methods
- User-friendly interface using **Streamlit**
- Supports **field, storage, and vertebrate pests**
- Search and filter pests by name, category, type, plant part attacked, crop stage affected, damage effect, and control method

---

## Pest Categories

1. **Field Pests** – e.g., Fall Armyworm, Maize Stalk Borer, Aphids  
2. **Storage Pests** – e.g., Maize Weevil, Larger Grain Borer, Flour Beetle  
3. **Vertebrate Pests** – e.g., Birds (Quelea quelea), Rodents  

Each profile includes a **description** of the pest and a **damage summary** for easy understanding.
